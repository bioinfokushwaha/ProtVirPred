# ParaVirPred

## AI-Based Parasite Virulence Prediction Web Server

ParaVirPred is a web-based bioinformatics platform developed at the National Institute of Animal Biotechnology (NIAB), Hyderabad, for the prediction of virulence-associated proteins from parasitic organisms using protein language models, physicochemical descriptors, and machine learning.

---

## Overview

Parasite virulence proteins play critical roles in host-pathogen interactions and disease progression. Experimental identification of virulence factors is time-consuming and expensive. ParaVirPred provides an AI-driven framework for rapid virulence prediction from protein sequences.

The platform integrates:

* ESM2 protein language model embeddings
* ProtT5 protein language model embeddings
* Physicochemical feature descriptors
* Machine learning classification models
* Interactive web-based prediction interface

---

## Key Features

### Single Sequence Prediction

Predict virulence potential from an individual protein sequence.

### FASTA Batch Prediction

Upload multi-sequence FASTA files and obtain predictions for all proteins simultaneously.

### Multiple Feature Combinations

Supported feature combinations include:

* ESM2
* ProtT5
* Physicochemical Features
* ESM2 + Physicochemical
* ProtT5 + Physicochemical
* ESM2 + ProtT5
* ESM2 + ProtT5 + Physicochemical

### Downloadable Results

Prediction outputs can be exported in CSV format.

## Installation

### Run Full App

```bash
pip install -r backend/requirements.txt
npm install --prefix frontend
python main.py --host 127.0.0.1 --port 8001
```

This builds the React frontend and serves it from the FastAPI backend on the same host and port.

To reuse an existing frontend build:

```bash
python main.py --host 127.0.0.1 --port 8001 --skip-frontend-build
```
